# base-react  
이 프로젝트는 기본적인 React 프로젝트 설정입니다.  
**수정날짜:** 2024-11-16

# Create
npx create-react-app base-react

# Start & Build
    npm start
    npm run build

# Install npm
    Axios (서버 통신)
        - npm install axios

    Redux (상태 관리)
        - npm install react-redux
        - npm i @reduxjs/toolkit 

    React Router (라우터)
        - npm i react-router-dom
